# Kayou

A complete, opinionated UI component library for [SolidJS](https://solidjs.com), built with TypeScript and Tailwind CSS.

📚 **Documentation:** [kayou-docs.vercel.app](https://kayou-docs.vercel.app)

## Packages

| Package | Description |
| --- | --- |
| [`@kayou/ui`](./packages/ui) | UI components (datatable, charts, rich-text editor, forms, …) |
| [`@kayou/hooks`](./packages/hooks) | Utility hooks and contexts |
| [`@kayou/icons`](./packages/icons) | Icon components |

## Getting started

```bash
pnpm install
pnpm dev          # run the docs site
pnpm build        # build all packages
pnpm test:e2e     # run Playwright tests
```
