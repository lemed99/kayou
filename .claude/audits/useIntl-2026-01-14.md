## API Surface

The returned `IntlShape` object provides:

- `formatMessage()` - Format translated messages
- `formatDate()` - Format dates according to locale
- `formatTime()` - Format times according to locale
- `formatNumber()` - Format numbers according to locale
- `formatRelativeTime()` - Format relative time strings
- `locale` - Current locale string
- And many more FormatJS methods
