## Troubleshooting

### Component not responding to clicks

**Problem:** Component appears but doesn't respond to interactions.

**Solution:** Check if `isDisabled` is true or if there's CSS blocking pointer events.

### Styles not applying

**Problem:** Custom classes don't seem to work.

**Solution:** The component uses `twMerge` for class composition. Make sure your classes have higher specificity if needed.

### TypeScript errors

**Problem:** Type errors when using the component.

**Solution:** Import the props type and ensure your props match the interface:

```typescript
import type { {ComponentName}Props } from '@exowpee/the_rock'
```
